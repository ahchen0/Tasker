TaskerGui module¶

class TaskerGui.Application(master=None)¶

Bases: tkinter.Frame

Main application class for the Tasker

Variables

  • master (Tk) – Parent class of application. Should be the tkinter root

  • subscribers – List of subscribers

  • menubar (TaskerMenuBar) – Menu bar of GUI

  • buttonbar (TaskerButtonBar) – Button bar of GUI

  • treeview (TaskerTreeView) – Satellite and point list of GUI

  • canvas (TaskerCanvas) – Map canvas of GUI

  • time (datetime) – The current time plotted in on the map

  • statusbar (TaskerStatusBar) – The status bar of the GUI. Displays the current time plotted

Parameters

master (Tk) – Parent class of TaskerGui. Usually the root.
