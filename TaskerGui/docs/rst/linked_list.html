class linked_list.linked_list¶

Bases: object

append(x=None)¶

Appends x, the linked_list_item to the end of the linked-list. If x is not a linked_list_item, it makes it one first. Always succeeds.

insert(index, x=None)¶

Insers x, the linked_list_item before an existing item in the linked-list, with position = index. If x is not a linked_list_item, it makes it one first. If there is not an item with the specified index, False is returned, other wise True is returned.

nodeat(i)¶

Returns node at index i. First node is index 0. Raises TypeError if i is not an integer Raises ValueError if i is not within the bounds of the list,

ie: 0 <= i < size

remove(x)¶

Removes x, the linked_list_item from the list Raises TypeError if x is not a linked_list_item Raises ValueError if linked-list is empty Returns True on success, False on failure
