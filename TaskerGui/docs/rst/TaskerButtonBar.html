TaskerButtonBar module¶

class TaskerButtonBar.TaskerButtonBar(master=None, size=32)¶

Bases: tkinter.Frame

Creates the buttonbar for use in the Tasker GUI. Builds each of the button items, and attaches functions to each.

Parameters

  • size (int) – The size of the button bar
