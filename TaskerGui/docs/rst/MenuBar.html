MenuBar module¶

class MenuBar.MenuBar(master=None)¶

Bases: tkinter.Frame

MenuBar allows creating menubars more easily by providing an “add_item” method that take a simple list to define one menu-bar heading. For example: file_menu = [“File”, “Open”, self.fileOpen, “quit”, self.master.destroy] self.menubar.item1=self.menubar.add_item(file_menu)

This can be called as often as needed to create all the items in a menubar.

Parameters

master (TaskerMenuBar) – parent class
