ButtonBar module¶

class ButtonBar.ButtonBar(master=None)¶

Bases: tkinter.Frame

ButtonBar allows creating button bars more easily by providing 2 functions for adding buttons: add_button() and add_icon(). For example:

self.buttonbar.add_button(“fileOpen”,self.fileOpen);

self.photo1 = tk.PhotoImage(file=”gear.gif”) self.buttonbar.add_icon(self.photo1,self.master.destroy)
